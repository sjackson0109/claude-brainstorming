## Overview

The Brainstorming Orchestrator is an intelligent system that automatically routes brainstorming requests to the most relevant specialised domain expertise. Rather than providing generic creative thinking, it leverages 150 specialised skills across technical, business, design, innovation, and strategic domains to deliver expert-level guidance tailored to your specific challenge.

## How It Works

1. **Intelligent Domain Detection** - Analyses your brainstorming request to identify relevant subject areas
2. **Strategic File Access** - Reads only the specific reference skills needed for your request  
3. **Expert Framework Application** - Applies domain-specific methodologies and best practices
4. **Integrated Multi-Domain Support** - Combines multiple specialisations when appropriate

## Usage Examples

**Simple Domain Request:**
> "Use your Brainstorming skill to support me with technical user interface design. Looking at aesthetics, pattern and colour irregularities within the UI, as well as font shape/size/weight inconsistencies."

**Result:** Routes to `user-interface-design.md` for expert UI design guidance including visual design systems, typography frameworks, and colour theory.

**Multi-Domain Request:**  
> "Help me brainstorm a digital transformation strategy that includes cybersecurity considerations and change management."

**Result:** Integrates expertise from `digital-strategy.md`, `cybersecurity.md`, and `change-management-psychology.md`.
