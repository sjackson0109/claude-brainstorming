# Quick Reference - Common Use Cases

## Domain-Specific Quick Requests

```
"Use your Brainstorming skill for [specific technical challenge]"
```

**Common Patterns:**
- **API Design**: "Help me brainstorm REST API security patterns for enterprise microservices"

- **DevOps**: "Help me design CI/CD pipelines for regulated financial services"

## Request Modifiers for Better Results

### Add Context
- **Industry**: "...in healthcare/finance/manufacturing"
- **Scale**: "...for startup/enterprise/global organisation" 
- **Timeline**: "...with 6-month/1-year/3-year horizon"
- **Budget**: "...within £X budget constraints"
- **Team Size**: "...with team of X people"

### Specify Depth
- **Strategic**: "...from strategic planning perspective"
- **Tactical**: "...with implementation focus"
- **Technical**: "...including technical architecture details"
- **Operational**: "...considering day-to-day operations"

### Request Format
- **Frameworks**: "...using established frameworks and methodologies"
- **Examples**: "...with specific examples and case studies"  
- **Steps**: "...with actionable steps and timeline"
- **Alternatives**: "...exploring multiple approaches and options"

## Quality Indicators in Responses

✅ **Good Response Includes:**
- Domain-specific terminology and frameworks
- Structured analysis with clear sections
- Practical implementation steps
- Risk assessment and mitigation strategies
- Multiple options and alternatives
- Integration across relevant domains

❌ **Request Refinement Needed If:**
- Response is too generic
- Missing domain-specific expertise
- Lacks practical implementation guidance
- Doesn't address stated constraints
- Missing multi-domain integration when needed

## Follow-up Question Patterns

### Drill Down
- "Let's explore the [specific aspect] you mentioned in more detail"
- "Focus on the [framework/approach] and provide implementation steps"
- "How would this apply specifically to [your context]?"

### Alternative Exploration
- "What other approaches could achieve similar outcomes?"
- "How would this differ in [alternative context/constraint]?"
- "What are the trade-offs between these options?"

### Integration
- "How do these recommendations work together systematically?"
- "What's the sequencing and dependencies between these approaches?"
- "How do I prioritise these options given my constraints?"
