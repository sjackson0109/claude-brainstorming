## 🚀 System Capabilities

### Intelligent Domain Detection
- Analyses request keywords and context
- Maps to appropriate reference expertise  
- Reads only relevant specialised files (typically 1-3)
- Applies domain-specific frameworks and methodologies

### Expert-Level Responses
- Uses established industry frameworks and best practices
- Provides structured analysis with implementation guidance
- Integrates insights across multiple domains when appropriate
- Includes risk assessment and mitigation strategies

### Efficiency Protocol
- **Strategic file access** - Reads only what's needed for your specific request
- **Progressive complexity** - Start broad, then drill down into specifics
- **Iterative refinement** - Build on previous responses for deeper insights
- **Context preservation** - Maintains awareness of constraints and requirements
