# Physical Security Assessment Brainstorming Skill

## Purpose

The Physical Security Assessment Brainstorming Skill specialises in comprehensive physical security evaluation strategy and facility protection assessment. This skill empowers strategic thinking through systematic physical security testing, access control evaluation, and comprehensive assessment methodologies for protecting physical assets, facilities, and personnel from unauthorised access and physical threats.

## Core Capabilities

### Comprehensive Physical Security Assessment Framework
- **Facility Security Architecture Evaluation**: Strategic assessment of physical security design including perimeter protection, access control systems, and surveillance infrastructure
- **Multi-Layer Physical Security Testing': Systematic evaluation of physical security controls across facility perimeters, building access, and sensitive area protection
- **Physical Threat Landscape Analysis': Comprehensive assessment of physical security threats including unauthorised access, theft, and facility compromise

### Access Control and Facility Protection Assessment
- **Physical Access Control System Testing': Strategic evaluation of badge readers, biometric systems, and electronic access control effectiveness
- **Perimeter Protection Assessment': Comprehensive testing of facility perimeters including fencing, barriers, and detection systems
- **Surveillance and Monitoring System Evaluation': Systematic assessment of video surveillance, alarm systems, and physical security monitoring capabilities

### Physical Penetration Testing and Security Validation
- **Authorised Physical Penetration Testing': Strategic testing of physical security controls through simulated unauthorised access attempts
- **Lock and Physical Barrier Assessment': Comprehensive evaluation of mechanical and electronic locking systems and physical barrier effectiveness
- **Physical Social Engineering Integration': Assessment of human factor vulnerabilities in physical security including tailgating and authority exploitation

## Implementation Framework

### Phase 1: Physical Security Assessment Planning (75 minutes)
- Conduct comprehensive facility security architecture analysis including site survey and threat assessment
- Develop physical security testing strategy based on facility type, security requirements, and risk profile
- Establish assessment methodology including safety protocols, legal compliance, and stakeholder coordination

### Phase 2: Systematic Physical Security Testing (95 minutes)
- Execute comprehensive access control testing including badge cloning, lock bypassing, and electronic system assessment
- Perform perimeter protection testing including barrier bypass, detection system evasion, and surveillance assessment
- Conduct authorised physical penetration testing with appropriate safety controls and legal authorisation

### Phase 3: Physical Security Architecture Evaluation (80 minutes)
- Analyse physical security system integration including access control, surveillance, and alarm system coordination
- Evaluate physical security monitoring and response capabilities including guard effectiveness and incident response
- Assess physical security policy implementation including visitor management and contractor access control

## Key Techniques and Methodologies

### Physical Security Testing Methodologies
- **ASIS International Guidelines Compliance': Systematic assessment approach using established physical security industry standards
- **Crime Prevention Through Environmental Design (CPTED) Evaluation': Strategic assessment of facility design for crime prevention and security enhancement
- **Layered Physical Security Assessment': Comprehensive evaluation of defence-in-depth physical security implementation

### Access Control and Surveillance Analysis
- **Electronic Security System Testing': Strategic evaluation of electronic access control, surveillance, and alarm systems
- **Mechanical Security Assessment': Comprehensive testing of locks, safes, and physical barrier effectiveness
- **Integration System Evaluation': Assessment of physical security system integration and centralized monitoring capabilities

### Physical Security Risk Assessment
- **Threat and Vulnerability Analysis': Systematic assessment of physical security threats and facility vulnerability factors
- **Asset Protection Evaluation': Strategic assessment of critical asset protection including data centres, executive areas, and sensitive facilities
- **Emergency and Crisis Response Assessment': Evaluation of physical security during emergency situations and crisis scenarios
