# CIS Critical Security Controls Brainstorming Skill

## Purpose

The CIS Critical Security Controls Brainstorming Skill integrates cybersecurity best practices into strategic thinking through systematic application of the centre for Internet Security (CIS) Critical Security Controls framework. This skill ensures that all strategic initiatives incorporate robust cybersecurity considerations, threat modeling, and defensive strategies aligned with internationally recognised security standards.

## Core Capabilities

### CIS Controls Framework Integration
- **Control Mapping Methodology**: Systematically map strategic initiatives against all 18 CIS Critical Security Controls
- **Control Prioritisation Analysis**: Apply CIS Implementation Groups (IG1, IG2, IG3) to align security investments with organisational maturity and risk profile
- **Control Effectiveness Assessment**: Evaluate proposed strategies through the lens of CIS control effectiveness metrics and measurements

### Security-First Strategy Development
- **Defensive Architecture Planning**: Integrate cybersecurity considerations into strategic architecture from inception
- **Threat-Informed Decision Making**: Incorporate current threat intelligence and attack patterns into strategic brainstorming sessions
- **Risk-Based Security Integration**: Balance security requirements with business objectives through systematic risk assessment

### Cybersecurity Governance Integration
- **Security Policy Alignment**: Ensure strategic initiatives align with organisational cybersecurity policies and CIS framework recommendations
- **Compliance Integration**: Incorporate cybersecurity compliance requirements into strategic planning processes
- **Security Metrics Integration**: Embed security performance measurement into strategic success criteria

## Implementation Framework

### CIS-Informed Brainstorming Process

**Phase 1: Security Context Assessment (45 minutes)**
- Map organisational cybersecurity maturity against CIS Implementation Groups
- Identify applicable CIS controls for the strategic domain under consideration
- Assess current threat landscape and relevant attack patterns
- Review existing cybersecurity governance and policy frameworks

**Phase 2: Security-Integrated Ideation (90 minutes)**
- Generate strategic alternatives while maintaining CIS control alignment
- Apply security-by-design principles to all proposed initiatives
- Integrate defensive thinking into creative problem-solving processes
- Consider security implications across all implementation phases

**Phase 3: Security Validation and Enhancement (60 minutes)**
- Validate strategic proposals against CIS control requirements
- Enhance strategies with additional security controls and safeguards
- Develop security-specific success metrics and monitoring approaches
- Create security escalation and incident response considerations

### CIS Controls Application Framework

**Basic CIS Controls (Implementation Group 1)**
- **Inventory and Control of Enterprise Assets (CIS Control 1)**: Ensure strategies include comprehensive asset identification and management
- **Inventory and Control of Software Assets (CIS Control 2)**: Integrate software lifecycle management into strategic technology decisions
- **Data Protection (CIS Control 3)**: Embed data protection and privacy considerations into all strategic initiatives

**Foundational CIS Controls (Implementation Group 2)**
- **Secure Configuration of Enterprise Assets and Software (CIS Control 4)**: Include security configuration management in technology strategy
- **Account Management (CIS Control 5)**: Integrate identity and access management into organisational strategy
- **Access Control Management (CIS Control 6)**: Ensure privilege management aligns with strategic organisational design

**Organisational CIS Controls (Implementation Group 3)**
- **Continuous Vulnerability Management (CIS Control 7)**: Include ongoing vulnerability assessment in strategic risk management
- **Audit Log Management (CIS Control 8)**: Integrate security monitoring and logging into operational strategy
- **Email and Web Browser Protections (CIS Control 9)**: Include communication security in digital transformation strategy
