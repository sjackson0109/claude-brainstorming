How It Works

Four intelligent steps to deliver expert-level brainstorming tailored to your specific challenge.

1

Intelligent Domain Detection

Analyses your brainstorming request to identify relevant subject areas automatically.

2

Strategic File Access

Reads only the specific reference skills needed for your request, keeping things focused and efficient.

3

Expert Framework Application

Applies domain-specific methodologies, frameworks, and best practices from specialised knowledge.

4

Integrated Multi-Domain Support

Combines multiple specialisations when your challenge spans more than one area of expertise.

Quick Installation

Get up and running in seconds with the Claude Code marketplace.

# Add the plugin from the marketplace
/plugin marketplace add sjackson0109/claude-brainstorming

# Install the brainstorming skill
/plugin install brainstorming@brainstorming-skills

That's it — the skill is now available in your Claude Code sessions.
